## Present Tense

## Standard verbs

## שלמים

| Form     | Hebrew example | Translation |
| -------- | -------------- | ----------- |
| singular | לוֹמֶדֶה       | They learn  |
| plural   | לוֹמְדִימוֹת   | They learn  |

## Verbs that end in heh

## ל’’ה

| Form     | Hebrew example | Translation |
| -------- | -------------- | ----------- |
| singular | עוֹשֶׂת        | They make   |
| plural   | עוֹשִׂימוֹת    | They make   |

## Verbs with an ayin as the middle letter of the root

## ע’’ו/ע’’י

| Form     | Hebrew example | Translation |
| -------- | -------------- | ----------- |
| singular | קָמֶה          | They rise   |
| plural   | קָמִימוֹת      | They rise   |
