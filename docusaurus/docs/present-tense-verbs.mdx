## Present Tense

## Standard verbs

## <HB> שלמים </HB>

| Form     | Hebrew example | Translation |
| -------- | -------------- | ----------- |
| singular | <HB> לוֹמֶדֶה </HB>     | They learn  |
| plural   |<HB> לוֹמְדִימוֹת </HB>  | They learn  |

## Verbs that end in heh

## <HB> ל’’ה </HB>

| Form     | Hebrew example | Translation |
| -------- | -------------- | ----------- |
| singular |<HB> עוֹשֶׂת </HB>       | They make   |
| plural   |<HB> עוֹשִׂימוֹת </HB>   | They make   |

## Verbs with a yud or vav as the second letter of the root

## <HB> ע’’ו/ע’’י </HB>

| Form     | Hebrew example | Translation |
| -------- | -------------- | ----------- |
| singular | <HB> קָמֶה </HB>          | They rise   |
| plural   | <HB> קָמִימוֹת </HB>     | They rise   |
